基于 Vue.js 开发的一套风格简洁的 UI 框架。


</div>


### 🌟 特点

- 提供简洁易用的 API 和清爽简洁的组件风格。
- 高覆盖率单元测试，保证组件可用性。
- 开箱即用的高质量 Vue 组件。


### 📦 安装

```bash
npm install coco-ui
```

```bash
yarn add coco-ui
```

### 🔨 示例

只支持按需加载组件：
```vue
import { Button } from 'coco-ui';
```
