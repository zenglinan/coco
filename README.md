<h2 align="center">基于 Vue.js 开发的一套风格简洁的 UI 框架。</h2>

### 📦 安装

```bash
npm install coco-ui
```

```bash
yarn add coco-ui
```

### 🔨 示例

```vue
import { Button } from 'coco-ui';
```
